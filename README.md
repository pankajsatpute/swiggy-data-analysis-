# swiggy-data-analysis-
End-to-end Swiggy restaurant data analysis using Python, SQL, and Power BI to explore pricing, customer ratings, delivery performance, and cuisine trends with interactive dashboards and business insights.
 Swiggy Data Analysis Project

#Project Overview
This project presents an end-to-end data analysis of Swiggy restaurant data to uncover insights related to pricing, customer ratings, delivery performance, and cuisine preferences.  
The goal is to demonstrate real-world Data Analyst skills using Python, SQL, and Power BI.

#Objectives
- Analyze restaurant distribution across cities
- Understand the relationship between price, ratings, and delivery time
- Identify top-performing cuisines
- Provide data-driven business recommendations
- Build an interactive Power BI dashboard

#Dataset Information
- Source: Swiggy restaurant dataset
- Records: ~8,600+
- Features include:
  - Restaurant Name
  - City & Area
  - Average Price for Two
  - Average Ratings
  - Total Ratings
  - Delivery Time
  - Food Type (Cuisine)

#Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn) – Data Cleaning & EDA
- MySQL – Data querying & analysis
- Power BI – Dashboard & visualization
- Excel / CSV – Data storage

#Data Cleaning & Preparation
- Removed duplicates and handled missing values
- Corrected data types
- Normalized cuisine data by extracting Primary Cuisine
- Removed outliers for ratings and delivery time
- Prepared analysis-ready dataset

#Exploratory Data Analysis (EDA)
Key analyses performed:
- City-wise restaurant distribution
- Price distribution and pricing trends
- Ratings distribution
- Delivery time performance analysis
- Cuisine popularity and performance

#Power BI Dashboard
The dashboard consists of 4 interactive pages

# Overview
- Total Restaurants
- Average Price
- Average Rating
- Delivery Time
- City-wise & cuisine-wise distribution

#Pricing & Ratings
- Price vs Average Rating (Scatter Plot)
- Average Price by City
- Top Restaurants Table

#Delivery Performance
- Delivery Time vs Ratings
- Delivery Time Distribution

#Cuisine Insights
- Top Cuisines by Popularity
- Average Rating by Cuisine
- Average Price by Cuisine

#Key Insights
- Faster delivery (≤ 40 minutes) leads to higher customer ratings
- Higher price does not always guarantee better ratings
- North Indian, Chinese, and Fast Food cuisines dominate the platform
- Moderate pricing performs better than premium pricing

#Business Recommendations
- Optimize delivery routes to reduce delivery time
- Promote high-rated and moderately priced restaurants
- Focus marketing strategies on popular cuisines
- Improve delivery efficiency to boost customer satisfaction
